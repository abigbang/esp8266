## Options:
* CrystalFreq:26M
* SPI SPEED: 40MHz
* SPI MODE: DIO
* FLASH SIZE: 32Mbit-C1
* BAUD: 115200

## Hardware connection:

DHT11|<---->|NODEMCU
-|-|-
Data|<---->| D1(GPIO5)

修改管脚，请修改以下地方：
1. app/include/driver/hal_temp_humd.h 10L: 
```C
#define DHT11_GPIO_PIN      5
```
2. app/driver/hal_temp_humd.c　200L:
```C
PIN_FUNC_SELECT(PERIPHS_IO_MUX_GPIO5_U, FUNC_GPIO5);
```
